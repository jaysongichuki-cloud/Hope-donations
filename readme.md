HOPE DONATIONS

A simple donation web application that allows users to donate to different causes

FEATURES

-Select different donation causes using interactive cards

-Make donations with amount and phone number 

-currency conversion

-Live donation tracking using local storage

-Recent donors list updates in real time

-Homepage statistics (total donations,donors,projects funded)

-Github Actions CI for automated testing (Jest)

TECK STACK

-HTML

-CSS

-JavaScript

-Jest (Unit Testing)

-Github Actions

-Exchange rate API
